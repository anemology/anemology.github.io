判斷兩個字串是否為 Anagrams (暫譯:變位詞)

Anagrams 的意思是兩個字串的組成是否相同

apple 和 pplea 會輸出 true

bird 和 biidd 則會輸出 false

解決方法

  1. 判斷兩個字串長度是否相同，若不同就可以直接輸出 false。
  2. 將字串分割成陣列。
  3. 將陣列依照字母順序排列。
  4. 比對陣列裡的每個元素是否相同。
